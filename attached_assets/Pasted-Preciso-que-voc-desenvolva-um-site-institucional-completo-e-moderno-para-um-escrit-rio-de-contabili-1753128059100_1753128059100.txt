Preciso que você desenvolva um site institucional completo e moderno para um escritório de contabilidade chamado Contabilize Assessoria Empresarial.

🧭 Objetivo do Projeto
Criar um site elegante, funcional e útil para empresários, autônomos e prestadores de serviço que buscam informação, serviços e suporte contábil confiável, com destaque para ferramentas práticas e um sistema inteligente de consulta de CNAE.

📌 Requisitos Gerais
Interface moderna, responsiva e com boa performance

Layout limpo, profissional e confiável (com destaque para o nome “Contabilize”)

Foco em navegação intuitiva, otimização SEO e acessibilidade

Sistema gerenciável (pode ser CMS ou admin customizável)

🧱 Seções / Páginas Essenciais
🏠 Página Inicial
Apresentação da Contabilize

Destaque dos serviços contábeis e diferenciais

CTA: "Fale com um contador", "Abrir empresa", “Consulta CNAE”

🧮 Serviços

MEI, ME, EPP, Lucro Presumido, Simples Nacional

Consultoria tributária

Atendimento PJ completo

Cada serviço com uma landing page própria e CTA para contato.

📊 Página de Consulta CNAE (diferencial principal!)
Ferramenta onde o usuário pode:

Pesquisar por palavra-chave (ex: "manicure", "consultoria", "salão", etc.)

Pesquisar por código CNAE ou parte dele (ex: “8412” ou “8412-4/00”)

Ao buscar, o sistema exibe:

"Ficou em dúvida? Fale com a Contabilize e escolha o melhor CNAE para pagar menos impostos."

⚙️ Tecnologias Sugeridas
Frontend: Next.js com TailwindCSS

Backend (caso a busca CNAE seja dinâmica): Node.js ou Python (FastAPI)

Banco de dados para armazenar as descrições CNAE: SQLite ou PostgreSQL

CMS (opcional): Strapi, Sanity ou CMS personalizado para edição de conteúdo

Hospedagem: Vercel (frontend), Railway/Render (backend)

API de CNAE: Se não for utilizar base própria, usar dados públicos da Receita + tratamento local
